#   地下避难所方案SubwayOfCivilianHousehold（未完成）
为满足战区平民的生命避难需求，尝试设计一套可以自助建设的「<mark>地下避难所</mark>」]方案。此方案适合30人左右的大家庭
<br/><sub>设计思路：向下挖掘「10米深的竖井」、再建造「阻击区」拒敌进入。阻击区的下侧设置「气封防爆门」防止烟熏水淹进「避难所」</sub>
<br/><sub>概览地址：<a target=_blank href='https://jackpanracc.github.io/SubwayOfCivilianHousehold/SubwayOfHousehold.htm'>https://JackPanrAcc.github.io/SubwayOfCivilianHousehold/SubwayOfHousehold.htm</a></sub>
<br/><sub>项目地址：github.com/JackPanrAcc/SubwayOfCivilianHousehold</sub>
<br/>
<br/>*** 重要提示 ***：</sub>
<br/>1、<b>通风口伪装</b>。通风口的隐蔽性是地道型避难所的最重要的考虑点。不具备隐蔽通风口的风险太大、考虑地道的特殊性不建议挖掘
<br/>2、<b>地下水情况</b>。地下50米内有大量流动地下水则不可执行。适度富含水份需要增加防水结构或控压结构，防水工程量太大太难不建议挖掘
<br/>3、<b>地层性质</b>。地下50米内有地质断裂带或流沙等结构则不可执行。地质结构太软需要增加额外的支撑件结构，支撑工程量太大太难不建议挖掘
<hr>关于地层性质&地下水：&nbsp;
<br/>1、上网查找当地地质资料或观察附近山势，了解地质断裂带
<br/>2、根据地表裸露情况和地质演化史，假设有哪些地层结构及均匀性。注：地质结构一般呈现条状、和断裂带平行
<br/>3、宅基地附近的水井、矿井、溶洞、等地点，去收集情况检验上述两条判断。例如水井在50米内能出水就不要挖了
<br/>4、在淘宝上采购「打井找水仪」。其原理是土壤岩石电阻远高于地下水。如果电阻很高肯定没水，如果电阻低常见的情况是地下水或泥浆
<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;假设有地下水平行于地质断裂带、避开电力和大型金属装置选采样点。把30厘米探针插入地下即可测量30米、50米等深处电阻
<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;电阻低则采购「手持钻井机」打探洞确认。少量地下水可在高水压方向用钢板或灌注混凝土形成防水墙结构。也可以采用抽水减压沟结构
<br/>5、加固结构建议用混凝土，严禁使用PVC等易燃材料。控水结构建议用抽水减压，同时兼顾监控水压水量、安排地道维护保养工作
<br/>
<br/>关于无人机：
<br/>1、「阻击区」内可以采用倾斜地面、水淹烟熏投掷爆炸物都会被流向缓冲区。「阻击区」还可以采用射击孔阻击敌人
<br/>2、阻击区主要战斗形态是fpv无人机<sub>a)钢筋撑渔网，并部署铁丝拒绝接触式爆破 b)多部署上述渔网降低无人机机动性 c)霰弹枪或榴弹发射器发射渔网 d)竖井内剪光纤遥控线</sub>
<br/>
<br/>关于战时负荷：
<br/>1、目前民用的标准主要是以色列1992年执行的《merkhav mugan 标准》，中国2006年执行的《人防甲类标准》。可用做参考
<br/>2、联合国曾记录某一周期内索马里有166起简易爆炸装置袭击，其中仅3起用到车载大型装置。推测爆炸当量大于80kgTNT占比小于5%
<br/>&nbsp;&nbsp;&nbsp;&nbsp;一般重要价值目标所用爆炸物的当量为5～80kgTNT。参考同期伤亡数超10人、人群聚集场所、车队军队等目标推断其数量约10-15%
<br/>&nbsp;&nbsp;&nbsp;&nbsp;整体来说个别地区80%小5kgTNT，95%小80kgTNT。
<br/>3、《merkhav mugan 标准》提到抗RPG 攻击，这是0.8kgTNT，在一般硬土中可以炸出0.8米深度、2米直径的坑
<br/>4、《人防甲类标准》的假想模式是空袭。核6的技术指标大约可以折算等于十米外50kgTNT的爆炸的冲击
<br/>5、0.8kgTNT 在一般硬土中可以炸出0.8米深度、2米直径的坑。也可以严重损毁埋深1.5米内的硬土地下室
<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;5kgTNT 在一般硬土中可以炸出2米深度、5米直径的坑。也可以严重损毁埋深3米内的硬土地下室
<br/>6、总结，<b>推荐避难所的埋深超3米</b>。没有条件则埋深至少超过1.5米，或是采购「预制式整体避难所」（这类避难所大约5w$，容纳2-5人）
<br/>
<br/>关于通风口：
<br/>1、「通风口伪装」没有万全之策，不要过份自信。千万不要过度依赖隐蔽性。否则你不如不要建避难所
<br/>2、配备制氧机或氧气发生药板，在氧气耗尽之前新挖掘新的逃生道可以实现逃生
<br/>&nbsp;&nbsp;&nbsp;&nbsp;普通人挖土约每小时1立方米土量。手工挖掘「应急逃生道工作间」距离地面可以是2米左右
<br/>&nbsp;&nbsp;&nbsp;&nbsp;如果有电动打井机辅助挖掘「应急逃生道工作间」距离地面可以是5米左右
<br/>
<br/>关于主出入口的伪装说明：
<br/>「手持钻井机」破土后用「吊篮」运走土壤，主出入口有大量土壤要运出。在战略上应当认定主出入口不具备保密性
<br/>地层性质不同，地质雷达的探地距离也不同。400MHz大约探地4米，200MHz大约探地9米，50MHz即便在复杂城市环境探地距离也超过10米。上述地质雷达均有单人携带的产品。所以，低防护避难所不适用高价值目标
