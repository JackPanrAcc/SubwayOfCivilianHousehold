# SubwayOfCivilianHousehold
<br/>https://github.com/JackPanrAcc/SubwayOfCivilianHousehold
<br/>为了满足在战区生活的 20人左右的家庭的生命避难需求，设计一款自助建设「<span style="color:red;">地道避难所方案</span>」
<br/>
<br/>可行性
<br/>1、<b>地层性质</b>。断裂带甚至流沙等地质结构不可行。地质结构太软时需要增加支撑件结构。地质结构太硬时需要使用挖掘机械
<br/>2、<b>地下水情况</b>。50米内有大量流动地下水则不可行。适度富水层需要增加防水结构，或是控压结构
<br/>
1. 颜色名：<span style="color: red;">红色文字</span> / <span style="color: blue;">蓝色文字</span>
2. 十六进制（精准）：<span style="color: #FF5733;">橙红色文字（#FF5733）</span>
3. RGBA（支持透明度）：<span style="color: rgba(0, 128, 0, 0.5);">半透明绿色文字</span>
