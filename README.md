# 地下避难所方案SubwayOfCivilianHousehold
<br/>为满足战区平民的生命避难需求，尝试设计一套可以自助建设的「<mark>地下避难所方案</mark>」]。此方案适合 20人左右的家庭。
<br/><sub>项目地址：http://github.com/JackPanrAcc/SubwayOfCivilianHousehold</sub>
<br/><sub>项目思路：竖直向下挖掘10米后，构筑「阻击区」防止人员进入。在阻击区后侧出口修建密封门防止水淹烟熏</sub>
<br/><sub>预览地址：<a target=_blank href='https://jackpanracc.github.io/SubwayOfCivilianHousehold/SubwayOfHousehold.htm'>https://jackpanracc.github.io/SubwayOfCivilianHousehold/SubwayOfHousehold.htm</a></sub>
<br/>
<br/>可行性
<br/>1、<b>通风口伪装</b>。通风口具有绝对隐蔽性是地道型避难所的前提条件。否则风险太大、不建议挖掘
<br/>2、<b>地层性质</b>。地质断裂带甚至流沙等结构不可执行。地质结构太软时需要增加额外的支撑件结构。地质结构太硬时需要使用挖掘机械
<br/>3、<b>地下水情况</b>。50米内有大量流动地下水则不可行。适度富水层需要增加防水结构，或是控压结构
<br/>
<br/>
<br/>a) 普通人在土质挖掘速度大约每小时 1立方米土量。一般游击性袭击不会超过 3小时，那么地下避难所距离地面 8米以上深度可以发挥作用
<br/>b) 「通风口伪装」不可能有万全之策。需要配备制氧机或氧气发生药板，在氧气耗尽之前重新挖掘也能逃生、避免被憋死
<br/>c) 应急逃生道可能的突破口是机密。注意保密
<br/>d) 挖掘的思路是用「手持圆盘钻井机」破土，用「吊篮」运走土壤
