# 地道避难所方案SubwayOfCivilianHousehold
<br/>为满足战区平民的生命避难需求，尝试设计一款可以自助建设的「<mark>地道避难所方案</mark>」]。此方案适合 20人左右的家庭。
<br/><sub>项目地址：http://github.com/JackPanrAcc/SubwayOfCivilianHousehold</sub>
<br/><sub>预览地址：<a target=_blank href='https://jackpanracc.github.io/SubwayOfCivilianHousehold/SubwayOfHousehold.htm'>https://jackpanracc.github.io/SubwayOfCivilianHousehold/SubwayOfHousehold.htm</a></sub>
<br/>
<br/>可行性
<br/>1、<b>地层性质</b>。断裂带甚至流沙等地质结构不可行。地质结构太软时需要增加支撑件结构。地质结构太硬时需要使用挖掘机械
<br/>2、<b>地下水情况</b>。50米内有大量流动地下水则不可行。适度富水层需要增加防水结构，或是控压结构
