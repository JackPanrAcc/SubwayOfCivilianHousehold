#   地下避难所/SubwayOfHousehold/Refuge/Shelter（未完成）
项目目标：为满足战区平民的生命避难需求，尝试设计一套<mark>低成本可自助建设的「地下避难所」</mark>方案。此方案适合30人左右的家庭
<br/>项目思路：向下挖掘「5米深的竖井」后构造「伏击区」拒敌进入。伏击区的后方设置「气封防爆门」可以防止烟熏水淹危及避难者
<br/><mark>重要提示</mark>：
<br/>1、<b>通风口伪装</b>。通风口的隐蔽性是地下避难所最重要的考虑点。不具备隐蔽排风口的地下避难所风险极大、建议选择其它避难方式
<br/>2、<b>地下水情况</b>。地下50米内有大量流动的地下水则不可执行。适度含水需要增加防水结构或控压结构，防水工程量太大太难不建议挖掘
<br/>3、<b>地层性质</b>。地下50米内有地质断裂带或流沙等结构则不可执行。地质结构太软需要增加支撑结构，支撑工程量太大太难不建议挖掘

<br/>设计和挖掘 的注意事项：
<br/>1、挖掘的误差控制
<br/>1.1、找朋友在B地手持灯光，并站在灯光后侧
<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;你在测绘地点A转动镜子直到朋友说镜子里看到了灯光，此时，镜子方向就是AB连线的方位角
<br/>1.2、此方案可以使用民用指南针达到0.5度左右的误差。定向挖掘50米后横向误差约4米
<br/>1.3、定向挖掘时，长度过长需要安排钻「通讯线孔」完成布线后掩埋。顺便校正地道的准确位置
<br/>
<br/>2、本方案所涉及的孔结构
<br/>2.1、「通讯线孔」是采用“直径0.1米的手持电动打井机”钻通布线后填实。可用于天线延长线、摄像头控制、无人机站控制、等
<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Huawei Mate 在700MHz应急频段可穿1米厚的废墟。我们需要把天线延长线送到地面半米处
<br/>2.2、「通风排气孔」是采用“直径0.1米的手持电动打井机”钻通的直孔。地表孔必须伪装，例如避免直对开放空间、适当掩饰热信号
<br/>2.3、「通风进气孔」是采用“直径0.1米的手持电动打井机”钻通的直孔
<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;从「进气孔」沿进气地道到「气泵房」布置空调风道，使用风扇泵送清洁空气
<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;从「气泵房」到「避难所空气浑浊处」布置空调风道，使用风扇泵送清洁空气
<br/>
<br/>3、本方案所涉及的伏击区的重要点位
<br/>3.1、「伏击区前线竖井」中轴线下端点是整个地道的测绘基准点，记为《<b>frontWell点</b>》
<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;frontWell距地表约10米。包含主出入口竖井4米、无防护集结区2米、伏击区前线竖井4米
<br/>3.2、「首席伏击岗哨」和frontWell点等高，记为《<b>theGuard点</b>》
<br/>3.3、「伏击区后方竖井」中轴线和frontWell、theGuard等高的地方，记为《<b>backWell点</b>》
<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;backWell 会安排通风孔和通讯线孔，其位置可以得到进一步校正
<br/>
<br/>4、建议「backWell 轴线」和「气泵房通道 轴线」相互可直接目视。降低定位的误差
<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;避难区范围内所有钻孔作业都将基于「气泵房通道 轴线」的精确位置
<br/>
<br/>5、挖掘工作均需评估地下水和地质结构：
<br/>5.1、上网查找当地的地质资料或观察附近山势。了解地质断裂带情况
<br/>5.2、根据裸露的地表和地质演化史，推测地层结构及走向。注：地质结构一般呈现条块状、和断裂带平行
<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;地层太松软、或是碎片化严重，要额外增加加固结构。建议用混凝土，不要用PVC等易燃材料
<br/>5.3、宅基地附近遇有水井、矿井、溶洞等都要收集资料，检验上述两条判断。例如有水井在50米内出水则不用考虑地下避难所
<br/>5.4、去淘宝采购「打井找水仪」。其原理是土壤岩石电阻远高于地下水。电阻很高肯定没水。如果电阻低、最常见的情况是地下水或泥浆
<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;假设有地下水平行于地质断裂带、避开电力和大型金属装置确定采样点。把30厘米探针插入地下可测6米/12米/20米/30米等处电阻
<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;电阻中用「手持电动钻井机」打探洞确认是否有水。水压大于0.3Mpa时地道设计必须交给专业人员，低于此值也建议找专业人员
<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;a)编制透水预案 b)编制地道维保制度，例如每天用电阻找水仪监测特定点位电阻 c)建抽水减压结构监测水情 d)重要位置使用隔水材质

<br/><del><sub>关于防爆负荷</sub></del>
<br/>7、联合国曾记录某一周期内索马里有166起简易爆炸装置袭击。其中仅3起用到车载大型装置，推测占比小于5%、爆炸当量大于80kgTNT
<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;一般重要价值采用的战术爆炸物当量5～80kgTNT。梳理其中伤亡数超10人、人群聚集场所、车队军队目标等情况，推断占比10-15%
<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;整体来说，民事爆炸案件中80%爆炸小5kgTNT，95%小80kgTNT
<br/>8、目前可供参考的民用标准主要是以色列1992年执行的《merkhav mugan 标准》，中国2006年执行的《人防甲类标准》
<br/>9、《merkhav mugan 标准》防爆要求是抗RPG 攻击，这是0.8kgTNT，在一般硬土中可以炸出0.8米深度、2米直径的坑
<br/>10、《人防甲类标准》的假想模式是空袭。核6技术指标大约折算等于十米外50kgTNT的爆炸的冲击
<br/>11、总结，0.8kgTNT 在一般硬土中可以炸出0.8米深度、2米直径的坑。也可以严重损毁埋深1.5米内的硬土地下室
<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;5 kgTNT 在一般硬土中可以炸出 2米深度、 5米直径的坑。也可以严重损毁埋深3米内的硬土地下室
<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;地下避难所的埋深应当超过3米，最少1.5米。否则建议<b>采购「预制式整体避难所」（约30w ¥。可容纳3人，进气孔被堵即投降）</b>
<br/>12、硬土中直径1米的竖井短于五米可以无需加固，单层埋深值建议定在5米以上。值太大可以酌情加固，值太小则丧失了优化的余地

<br/>13、伏击区侧面结构 见下图
<img width='300px' src='https://jackpanracc.github.io/SubwayOfCivilianHousehold/photoForAmbushDesign.jpg'></img>
<br/>13.1、frontWell轴和「伏击区斜坡」交汇点记为《伏击区中心点/AmbushRoomCenter》
<br/>13.2、斜坡步行道的宽度推荐0.3米。需要错开frontWell竖井。施工时有大型珩架。平时放置小型的辅助码头，伏击前及时撤走辅助码头
<br/>13.3、步行道下侧斜坡斜率推荐45度。参考弹坑的坑壁稳定角，粉质土约35度，粘土约45度，岩石类约60度
<br/>13.4、《伏击区最低平面》距离《下侧短斜面最高点》垂直距离最少1米。避免底部爆炸的回填土垫高底部
<br/>13.5、《伏击区最低平面》距离《AmbushRoomCenter》垂直距离最少3米。避免敌人滚落斜坡后，站立破坏AmbushRoomCenter
<br/>13.6、步行道上侧斜坡垂直高度推荐2.5米。避免敌人在步行道上站立破坏伏击射击口
<br/>13.7、步行道上侧斜坡斜率。要确保伏击人员可以扫射伏击区内任意位置，且方便平时步行
<br/>
<br/>14、采用上述倾斜设计实现无死角扫射，首席伏击岗哨设置在伏击区的左上角。射击口大小酌情50φ30
<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;主伏击岗后侧是一个竖井。若有手雷进入战壕则丢进竖井
<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;主伏击岗附近设灯光照向伏击区的屋顶，通过反射光进行照明
<br/>15、针对frontWell 竖井内的敌人设置骚扰哨位。射击口大小酌情80x35x5
<br/>16、在AmbushRoomCenter正下方设置狙击哨位，单兵防守时此位置可放置遥控步枪。射击孔大小酌情200φ10
<br/>
<br/>17、上述结构中一般敌人不会贸然追击。有效的战斗形态主要包括fpv无人机+机器狗、烟熏水淹、封闭地道、钻孔进入特定地点攻击、等
<br/>18、「伏击区」采用倾斜地面，烟熏水淹投掷爆炸物都会被引导去「伏击区最低部」。
<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;伏击区步行道以下空间，多运一立方土壤，在遇到水淹时就会多一分钟时间撤离
<br/>19、封闭地道、爆破和水淹在战术效用是一样的
<br/>20、钻孔进入特定地点攻击，可能是手持钻孔机，也可能是JADM。预案成本太高没必要。遇见时用冲锋枪把人赶走、再把孔堵塞

<br/>21、反无人机 = a)钢筋撑起渔网、并加装铁丝拒绝接触式爆破 b)榴弹发射渔网 c)霰弹枪 d)无线干扰枪+竖井内剪光纤
<br/>22、防强攻渔网。渔网固定在钢筋前端张起、贴合住伏击区后方走廊。上两根钢筋可远程解绑、渔网下垂后用下侧钢筋拖回修补
<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;在「伏击区后方走廊」外设置渔网维修区。存储五套渔网、钢筋、以及补渔网所需小铁丝等物资
<br/>23、防逃脱渔网。frontWell位置做三、四个挂钩。从战壕水平伸出竹竿，顶端是直径1.2米的圆形鱼网，卡住后可以防无人机逃脱
<br/>24、剪光纤。骚扰哨的射击口是0.35米的宽形口子，找个铁钩子可以伸进去尝试捞光纤

<br/>安全保障性 的注意事项：
<br/>25、<mark>「冗余的地道」是提高安全系数的唯一正确途径</mark>
<br/>26、配备制氧机和氧气发生药板。在氧气耗尽之前新挖掘新的逃生道完成逃生
<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;普通人一小时大约挖掘1立方米土量。《应急性挖掘的长度》手工挖掘时建议2米左右，能确保障电动钻井机使用则可以更长
<br/>27、主出入口竖井很难伪装。「手持电动钻井机」破土后用「吊篮」运走土壤。主出入口有大量土壤运出，在战略上应当认定不具备保密性
<br/>28、雷达的探地深度不同：400MHz大约探地4米，200MHz大约探地9米，50MHz即便在复杂城市环境探地距离也超过10米
<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;各波段雷达产品均有单人携带的产品。遇到强大的敌人则假定其准确知晓地道走向。可见，低成本避难并不适宜保护高价值目标
