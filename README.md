# SubwayOfCivilianHousehold
<br/><samll>httpithub.com/JackPanrAcc/SubwayOfCivilianHousehold</small>
<br/>为了满足在战区生活的 20人左右的家庭的生命避难需求，啊<small>（注：小号）</small>设计一款自助建设「<mark>地道避难所方案</mark>」]
<br/>
<br/>可行性
<br/>1、<i>地层性质</i>。断裂带甚至流沙等地质结构不可行。地质结构太软时需要增加支撑件结构。地质结构太硬时需要使用挖掘机械
<br/>2、<b>地下水情况</b>。50米内有大量流动地下水则不可行。适度富水层需要增加防水结构，或是控压结构
